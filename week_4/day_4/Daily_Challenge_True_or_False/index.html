Fonction allTruthy()

La fonction allTruthy() vérifie si tous les paramètres passés sont des valeurs "truthy" (évaluées à true dans un contexte booléen).

En JavaScript, les valeurs "falsy" sont : false, 0, "", null, undefined, NaN. Toutes les autres valeurs sont "truthy".

Implémentation de la fonction

function allTruthy(...args) {
    // Vérifie si tous les éléments du tableau sont truthy
    return args.every(arg => Boolean(arg));
}

Exemples d'utilisation

allTruthy(true, true, true)
➞ true
allTruthy(true, false, true)
➞ false
allTruthy(5, 4, 3, 2, 1, 0)
➞ false
allTruthy('hello', 1, [], {})
➞ true

Autres implémentations

// Version avec boucle for
function allTruthyFor(...args) {
    for (let i = 0; i < args.length; i++) {
        if (!args[i]) {
            return false;
        }
    }
    return true;
}

// Version avec Array.reduce()
function allTruthyReduce(...args) {
    return args.reduce((acc, curr) => acc && Boolean(curr), true);
}
